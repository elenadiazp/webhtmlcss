PÁRRAFOS

Los párrafos se hacen con la etiqueta p.

HTML (Lenguaje de Marcas de Hipertexto, del inglés HyperText Markup Language) es el componente más básico de la Web. Define el significado y la estructura del contenido web. Además de HTML, generalmente se utilizan otras tecnologías para describir la apariencia/presentación de una página web (CSS) o la funcionalidad/comportamiento (JavaScript).

Como vemos, entre cada párrafo hay una línea en blanco. Si queremos hacer un salto de línea sin dejar ninguna línea en blanco utilizamos la etiqueta br.

Vamos a ver otras etiquetas

Para poner la letra en negrita utilizamos la etiqueta strong. Esta en negrita.

Para poner la letra en cursiva utilizamos la etiqueta em. Esto está en cursiva.

Para relaizar subíndices y superíndices utilizamos las etiquetas sub y sup respectivamente:
